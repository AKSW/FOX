How to use FOX

FOX accepts HTTP POST requests to: http://{{host}}/api and http://{{host}}/api/ner/feedback

Parameter /api

Optional parameters and there default values are marked.

  • input : text or an url
  • type : { text | url }
  • task : { NER }
  • output : { JSONLD | N3 | N-TRIPLE | RDF/{ JSON | XML | XML-ABBREV} | TURTLE }
  • nif : { true | false }
  • returnHtml : { true | false }
  • foxlight : { true | false }

Response /api

Each value is urlencoded. 
[{
    "input"  : "", 
    "output" : "",
    "log"    : ""
}]

Example requests with curl

/api

Content-Type: application/x-www-form-urlencoded
curl -d type=text -d task=NER -d output=JSONLD --data-urlencode "input=The foundation of the University of Leipzig in 1409 initiated the city's development into a centre of German law and the publishing industry, and towards being a location of the Reichsgericht (High Court), and the German National Library (founded in 1912). The philosopher and mathematician Gottfried Leibniz was born in Leipzig in 1646, and attended the university from 1661-1666." -H "Content-Type: application/x-www-form-urlencoded" http://{{host}}/api > response.txt

curl -d type=url -d task=NER -d output=TURTLE -d foxlight=true --data-urlencode "input=http://en.wikipedia.org/wiki/Leipzig_University" -H "Content-Type: application/x-www-form-urlencoded" http://{{host}}/api > response.txt

Content-Type: application/json
curl -d @request.txt -H "Content-Type: application/json" http://{{host}}/api > response.txt
request.txt 
{
    "input": "Cologne German: Köln, Kölsch: Kölle is Germany's fourth-largest city (after Berlin, Hamburg, and Munich), and is the largest city both in the German Federal State of North Rhine-Westphalia and within the Rhine-Ruhr Metropolitan Area, one of the major European metropolitan areas with more than ten million inhabitants. Cologne is located on both sides of the Rhine River. The city's famous Cologne Cathedral (Kölner Dom) is the seat of the Catholic Archbishop of Cologne. The University of Cologne (Universität zu Köln) is one of Europe's oldest and largest universities.",
    "type": "text",
    "task": "ner",
    "output": "TURTLE"
}
