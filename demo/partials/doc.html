How to use

web API
FOX web API accepts HTTP POST requests also in json format to: http://{{host}}/api

parameter
  • input : text or an url
  • type : { text | url }
  • task : { NER }
  • output : { JSONLD | N3 | N-TRIPLE | RDF/{ JSON | XML | XML-ABBREV} | TURTLE }
  • nif : { true | false }

Example requests with curl

Content-Type: application/x-www-form-urlencoded
curl -d type=text -d task=NER -d output=JSONLD --data-urlencode "input=The foundation of the University of Leipzig in 1409 initiated the city's development into a centre of German law and the publishing industry, and towards being a location of the Reichsgericht (High Court), and the German National Library (founded in 1912). The philosopher and mathematician Gottfried Leibniz was born in Leipzig in 1646, and attended the university from 1661-1666." --header "Content-Type: application/x-www-form-urlencoded" http://{{host}}/api
curl -d type=url -d task=NER -d output=TURTLE --data-urlencode "input=http://en.wikipedia.org/wiki/Leipzig_University" --header "Content-Type: application/x-www-form-urlencoded" http://{{host}}/api
